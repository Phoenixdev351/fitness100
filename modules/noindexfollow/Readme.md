################### SEO NOindex,follow module for Prestashop ###################

#### About

SEO NOindex,follow module gives you the controls to get rid of duplicate content with INDEX/ NOINDEX and FOLLOW/ NOFOLLOW options for each page. Now you don't need to use mass indexing for unnecessary pages in PrestaShop.

###### Features:

    -  Avoid duplicate category paginations for same category by adding canonical url.
	-  Optimize search results for all search engines
	-  Get rid of mass indexing for unnecessary pages.
	-  Set FOLLOW/ NOFOLLOW option for each page.
	-  Set INDEX/ NOINDEX option for each page.
	-  Ultimate tool to get rid of duplicate content.
	-  Gives you more controls over each page content.
	-  List of all Categories, Products, CMS, CMS Categories, Manufacturers, Suppliers, Default and Modules pages for INDEX & FOLLOW options.
	-  Update your robots.txt file directly from module settings.

 
NOTE - for PS1.4 versions only:
- Your theme’s header.tpl file should be writable. After installation please crosscheck following line in it:
<meta name="robots" content="{if isset($nobots)}no{/if}index,{if isset($nofollow) && $nofollow}no{/if}follow" />  
It should be there if not then please copy it manually.

#### The benefits for customers:

With this module you can set INDEX/ NOINDEX and FOLLOW/ NOFOLLOW options for each page separately in Prestashop. It list out all CMS, Default and Modules pages to set these options. Days are over for mass indexing with this module. With no duplicae content in your shop, it optimizes the results from all search engines and that results in moving up the ranking.
 
#### Installation:

USE Add modules button to upload the ZIP file of module. After uploading click on install button to install it.
After installation you can see the list of all the CMS, Default and Module pages on the configuration page. There you can set the INDEX/ NOINDEX and FOLLOW/ NOFOLLOW options for each page separately. After that press Save button and you’re done.
